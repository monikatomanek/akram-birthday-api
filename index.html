🎂 Akram Mukasa Birthday API

A small birthday REST API, built as a slightly unnecessary but very sincere gift for a legendary coding tutor.

It includes wisdom, cake reports, version-based age protection, and light debugging support - all assembled with appreciation, birthday energy, and perhaps questionable efficiency.
